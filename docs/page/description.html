ist ihr Passwort sicher? Das erfahren sie hier!
  1. Ist ihr Passwort mindestens 8 Ziffern Lang?
  2. Behinhaltet ihr Passwort Buchstaben und Zahlen?
  3. Verwenden sie ein Sonderzeichen wie bspw. #,%,@?
  4. Und verwenden sie für jede Anmeldung ein anderes Passwort?
Wenn sie eine dieser generellen Vorgaben nicht erfüllen können, sind sie NICHT sicher im Internet unterwegs. In diesem Fall können sie bei uns so viele Passwörter wie sie wollen erstellen und sich im Internet weiterhin schützen.
